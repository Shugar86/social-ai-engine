# Workflow одобрения через Telegram

## Роли

- **Admin** — полный доступ: одобрение, редактирование, отклонение, экстренная публикация, настройка проекта
- **Editor** — одобрение, редактирование, отклонение. Без экстренной публикации и настроек.

## Поток

### 1. Уведомление о новом посте

Бот шлёт сообщение в чат/группу:

```
🆕 Новый пост на одобрение

📋 Проект: doctor_yulia
📝 Тип: myth_bust
🎨 Формат: carousel (4 слайда)
📅 Запланировано: 01.06.2025 09:00 MSK

─── Caption ───
Миф: SPF 50 защищает в 2 раза лучше SPF 30.
Факт: разница всего 1%...

─── Медиа ───
[📎 слайд_1.png] [📎 слайд_2.png] [📎 слайд_3.png] [📎 слайд_4.png]

[✅ Одобрить] [✏️ Изменить текст] [❌ Отклонить] [🚀 Опубликовать сейчас]
```

### 2. Действия

| Кнопка | Действие | Кто |
|--------|----------|-----|
| ✅ Одобрить | `status → APPROVED`. Пост уйдёт в публикацию по `scheduled_at`. | Admin, Editor |
| ✏️ Изменить текст | Бот просит прислать новый caption. После получения — показывает превью снова. | Admin, Editor |
| ❌ Отклонить | `status → REJECTED`. Тема возвращается в пул. Бот спрашивает причину (опционально). | Admin, Editor |
| 🚀 Опубликовать сейчас | `status → APPROVED`, игнорирует `scheduled_at`, публикует немедленно. | Admin only |

### 3. Редактирование

После нажатия ✏️:

```
✏️ Режим редактирования

Пришлите новый текст caption'а.
Текущий:
---
Миф: SPF 50 защищает в 2 раза лучше SPF 30.
Факт: разница всего 1%...
---

Для отмены пришлите /cancel
```

Юзер присылает новый текст → бот показывает превью → кнопки ✅ ❌ снова.

### 4. Статусы и уведомления

| Событие | Уведомление |
|---------|-------------|
| Пост опубликован | "✅ Опубликовано: https://instagram.com/p/..." |
| Пост отклонён | "❌ Отклонено: doctor_yulia / myth_bust / SPF" |
| Ошибка публикации | "⚠️ Ошибка публикации: <reason>. Пост возвращён на доработку." |
| Автопилот включён | "🤖 Автопилот: пост doctor_yulia / tip_of_day опубликован автоматически." |

### 5. Команды бота

| Команда | Описание | Кто |
|---------|----------|-----|
| `/status` | Список постов в очереди по проектам | Admin, Editor |
| `/projects` | Список проектов и их статус | Admin, Editor |
| `/toggle <project>` | Переключить manual ↔ auto | Admin |
| `/skip <post_id>` | Пропустить пост (отменить публикацию) | Admin |
| `/now <post_id>` | Опубликовать сейчас | Admin |

### 6. Режим автопилота

Когда `approval.mode = "auto"`:
- Бот не шлёт превью на одобрение
- Посты публикуются по `scheduled_at` автоматически
- Бот шлёт уведомление *после* публикации: "🤖 Автопилот: опубликовано..."
- Admin может в любой момент нажать 🛑 в уведомлении и отключить автопилот для проекта

## Технические детали

- Сообщения с превью хранят `message_id` → при одобрении редактируем сообщение, добавляя ✅ в начало
- Медиа хранятся временно в `uploads/<project>/<post_id>/`, удаляются через 7 дней после публикации
- Для каруселей Telegram ограничивает 10 фото в одном сообщении — если больше, шлём альбомом
