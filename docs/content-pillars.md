# Контент-стратегия

## Проект: doctor_yulia

### Бренд-голос
- Экспертный, но не занудный
- Дружелюбный, но профессиональный
- Честный (не льстит, но и не пугает)
- Без медицинских диагнозов и назначений — только образование

### Контентные столпы

| Столп | Частота | Форматы | Цель |
|-------|---------|---------|------|
| **Мифы vs Факты** | 2×/нед | carousel, avatar_video | Развенчать популярные заблуждения, показать экспертность |
| **Совет дня** | ежедневно | tip_image, avatar_video | Повседневная полезность, узнаваемость |
| **Процедуры: до/после** | 1×/нед | carousel, Reels | Демонстрация результата (только с согласия, без указания личности) |
| **За кулисами** | 1×/нед | avatar_video, Stories | Личность врача, процесс работы, оборудование |
| **Ответы на вопросы** | 1×/нед | carousel, Reels | FAQ из комментариев и DM |
| **Тренды: разбор** | 1×/нед | avatar_video | Актуальность, привлечение трафика по трендовым хэштегам |

### Запрещённый контент
- До/после без согласия клиента
- Сравнение с конкурентами
- Обещания конкретного результата ("уберём морщины на 100%")
- Назначение лечения в постах
- Паникёрские заголовки ("Это убивает вашу кожу!")

### Пример content-plan.yaml

```yaml
plan:
  - date: "2025-06-01"
    topic: "Почему SPF 50 не лучше SPF 30"
    type: "myth_bust"
    format: "carousel"
  - date: "2025-06-02"
    topic: "Утренняя рутина для сухой кожи"
    type: "tip_of_day"
    format: "avatar_video"
  - date: "2025-06-03"
    topic: "Что происходит с кожей после 30"
    type: "educational"
    format: "carousel"
```

## Проект: ai_stylist

### Бренд-голос
- Модный, дерзкий, но доступный
- AI-технологичный, но не роботизированный
- Визуальный — картинки важнее текста

### Контентные столпы

| Столп | Частота | Форматы | Цель |
|-------|---------|---------|------|
| **AI-анализ: примеры** | 3×/нед | score_card_image, carousel | Демонстрация возможностей приложения |
| **Совет стилиста** | ежедневно | tip_image, Reels | Полезность, трафик |
| **Тренды недели** | 2×/нед | carousel, Reels | Актуальность, хайп |
| **Цветотипы** | 2×/нед | carousel, score_card_image | Образовательный контент, вирусность |
| **User-generated** | 1×/нед | Reels, Stories | Социальное доказательство |
| **За кулисами AI** | 1×/нед | carousel | Прозрачность, доверие к технологии |

### Вирусные механики

1. **"Узнай свой цветотип"** — пост с каруселью, в последнем слайде CTA: ссылка на бот/приложение
2. **"AI vs Реальный стилист"** — сравнение советов, вызывает обсуждение
3. **"До/после стиля"** — одна и та же девушка, AI подобрал образ → визуальный контраст

### Reuse существующих ассетов

- `card_engine.py` уже генерирует 1080×1080 карточки — используем как есть
- `analyzer.py` уже даёт JSON-репорт — можно парсить для генерации caption'ов
- Цветовые палитры из репорта — отличный визуальный контент для каруселей
